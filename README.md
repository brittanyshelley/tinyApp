## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session
- morgan

## Getting Started

- Clone project locally in code your editor prior to installing dependencies.
- Install all dependencies (using the `npm install` command).
- Run the development web server using the `npm start` command.
